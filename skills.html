C

Desciprtion

- A powerful, low-level programming language that serves as the foundation for many modern programming languages and operating systems. Known for its performance and efficiency, C is widely used for developing system software, embedded systems, and applications requiring direct hardware interaction.

Java

Desciprtion

- Object-oriented programming language known for its “write once, run anywhere” capability. It is widely used for developing cross-platform applications, from web and mobile apps to enterprise-level software and backend systems.

Python

Desciprtion

x86/x64 ASM

Desciprtion

- A low-level programming language that provides direct control over hardware and system resources. It is essential for tasks such as performance optimization, reverse engineering, and understanding the inner workings of processors.

~ Software and Tools ~

Ida-Pro

Desciprtion

- A powerful disassembler and debugger widely used in reverse engineering and cybersecurity. It translates machine code into human-readable assembly language, enabling analysis of software binaries.

Re-Class

Desciprtion

- A memory analysis and reverse engineering tool focused on inspecting and modifying data structures in a running program. It provides a graphical interface for mapping memory regions, allowing users to identify class hierarchies, data structures, and offsets.

Cheat Engine

Description

- An open-source tool designed for memory scanning and modification, often used in game hacking to manipulate in-game values such as health, currency, or experience points. It includes features like memory inspection, pointer scanning, and code injection. Cheat Engine can be used for educational purposes and debugging applications by analyzing how programs manage memory and respond to changes.

Windows API

Description

- A comprehensive set of functions and tools provided by Microsoft to enable developers to interact directly with the Windows operating system. It enables tasks such as managing system resources handling files, enabling network communication, and manipulate memory.
