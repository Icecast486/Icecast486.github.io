Overview

Technologies and Tools

- C++ - C - Cheat Engine - IDA Pro - Re-Class - Ghidra - Function Hooking (min-hook) - Windows API

Features

- Aimbot - God Mode - Infinite Ammo - Infinite Armor - No Recoil

Why and Skills Aquired

- This trainer (or cheat) is a way for me to practice my reverse engeneering skills. I love to play video games and recently, I've thought about what actaully goes on under the hood when I play video games. I read some articlles about how to use Cheat Engine's memory scanner, debugger, and code injector.
